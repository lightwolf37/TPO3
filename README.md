# Отчёт по тестированию ПО — Веб-приложение “ToDo-лист”

## 1. Введение
**Цель работы:**  
Провести системное тестирование веб-приложения “ToDo-лист” для проверки корректности работы функционала, выявления дефектов и оценки соответствия требованиям.  

**Краткое описание программы:**  
Приложение позволяет пользователю создавать, редактировать, удалять задачи, отмечать их как выполненные, фильтровать по статусу, а также отображает все задачи в списке.  
Ссылка на программу/сайт/приложение

**Обоснование тестирования:**  
Системное тестирование необходимо для оценки поведения приложения с точки зрения конечного пользователя, проверки соответствия требованиям (SRS), интерфейса и производительности.

---

## 2. Тест-план (Test Plan)

### 2.1 Цели и задачи
- Проверка корректности добавления, редактирования и удаления задач  
- Проверка работы фильтров и статусов задач  
- Проверка корректного отображения элементов интерфейса  
- Выявление ошибок и дефектов  
- Проверка соответствия требованиям спецификации

### 2.2 Область применения
**Тестируемые модули:**  
- Модуль добавления задач  
- Модуль редактирования задач  
- Модуль удаления задач  
- Модуль отметки задач как выполненных  
- Модуль фильтрации по статусу  

**Ограничения:**  
- Тестирование проводится только на десктопной версии в Chrome  
- Не тестируются мобильные версии и альтернативные браузеры  

### 2.3 Стратегия тестирования
- Функциональное тестирование всех модулей  
- Регрессионное тестирование после исправления багов  
- Тестирование GUI для проверки корректного отображения интерфейса  
- Smoke-тестирование при каждом запуске приложения  

### 2.4 Критерии начала тестирования
- Приложение запускается в браузере без ошибок  
- Реализован основной функционал (CRUD операций с задачами)  

### 2.5 Критерии завершения тестирования
- Выполнено ≥ 90% тест-кейсов  
- Все критические баги зарегистрированы и описаны  
- Программа соответствует требованиям SRS в основной функциональности  

### 2.6 Тестовая среда
- **ОС:** Windows 10 Pro (64-bit)  
- **Браузер:** Chrome v112  
- **Сервер:** локальный сервер Node.js v18  
- **Дополнительно:** интернет-соединение, редактор кода, утилита для снятия скриншотов
