# Отчёт по тестированию ПО — Веб-приложение “cosmos-online.ru”

## 1. Введение
**Цель работы:**  
Провести системное тестирование веб-приложения “cosmos-online.ru” для проверки корректности работы функционала, выявления дефектов и оценки соответствия требованиям.  

**Краткое описание программы:**  
Веб-приложение предназначено для поиска информации по введённому пользователем запросу. Пользователь может вводить ключевые слова, получать список результатов, переходить по найденным элементам и переходить по тегам.
Ссылка на приложение: [https://cosmos-online.ru/search]

**Обоснование тестирования:**  
Тестирование необходимо для проверки корректности работы поиска, отображения результатов, обработки пользовательского ввода, а также выявления ошибок интерфейса и логики.

---

## 2. Тест-план (Test Plan)

### 2.1 Цели и задачи
- Проверка корректности работы поиска
- Проверка обработки пользовательского ввода (пустой ввод, спецсимволы и т.д.)
- Проверка отображения результатов поиска
- Проверка перехода по результатам
- Выявление ошибок и дефектов
- Проверка удобства пользовательского интерфейса и корректность её
### 2.2 Область применения
**Тестируемые модули:**  
- Модуль ввода поискового запроса
- Модуль обработки запроса
- Модуль отображения результатов поиска
- Модуль интерфейса 
**Ограничения:**  
- Тестирование проводится только на десктопной версии в Chrome  
- Не тестируются мобильные версии и альтернативные браузеры  

### 2.3 Стратегия тестирования
- Функциональное тестирование поиска
- Тестирование граничных значений (пустой ввод, длинные строки)
- Тестирование GUI (интерфейса поиска)
- Smoke-тестирование (проверка базовой работоспособности страницы)
- Регрессионное тестирование после исправления ошибок
### 2.4 Критерии начала тестирования
- Страница поиска загружается без ошибок
- Поле ввода и кнопка поиска отображаются
- Доступен ввод текста
### 2.5 Критерии завершения тестирования
- Выполнено не менее 90% тест-кейсов
- Все критические ошибки задокументированы
- Поиск корректно обрабатывает основные пользовательские сценарии

### 2.6 Тестовая среда
- **ОС:** Windows 10 Pro (64-bit)  
- **Браузер:** Chrome v112  
- **Сервер:** локальный сервер Node.js v18  
- **Дополнительно:** интернет-соединение, редактор кода, утилита для снятия скриншотов  
## 2.7 Виды тестирования, применённые в работе
В рамках системного тестирования веб-приложения были использованы следующие виды тестирования:
- **Функциональное тестирование** – проверка корректности работы поиска, переходов и медиа-контента
- **Тестирование графического интерфейса (GUI)** – проверка отображения элементов интерфейса (кнопки, поля, изображения)
- **Smoke-тестирование** – базовая проверка работоспособности страницы поиска
- **Регрессионное тестирование** – повторная проверка после выявления ошибок
- **Юзабилити-тестирование** – оценка удобства использования поиска и навигации
## 2.8 Типы требований и покрытие тестами
### Функциональные требования:
- Поиск должен отображать результаты по введённому запросу (TC-02)
- Переход по результатам должен открывать соответствующую страницу (TC-06)
- Видео и аудио должны воспроизводиться (TC-07, TC-10)
### Нефункциональные требования:
- Поиск должен корректно обрабатывать пустой ввод (TC-03)
- Система должна обрабатывать специальные символы (TC-05)
- Страница должна загружаться без ошибок (TC-01)
### Пользовательские требования:
- Интерфейс должен быть понятным и удобным
- Пользователь должен легко находить поиск и навигацию
- Контент должен корректно отображаться
## 2.9 Тестовые данные
В процессе тестирования использовались следующие типы данных:
- **Корректные данные:**  
космос, инопланетяне, планеты
- **Некорректные данные:**  
"" (пусто), "!@#$%^&*"
- **Граничные значения:**  
длинные строки (>100 символов)
## 2.10 Анализ результатов тестирования
- Всего выполнено: **11 тест-кейсов**
- Успешно пройдено: **7**
- Провалено: **4**
 **Процент успешности:** ~64%
### Основные проблемы:
- Ошибки загрузки контента (изображения, аудио)
- Некорректная обработка пользовательского ввода
- Нестабильная работа поиска

## 3. Test Suite

### Test Suite: Поиск по сайту
- **TC-02 Ввод текста в поле поиска** – проверка возможности ввода запроса.
- **TC-03 Поиск с пустым вводом** – проверка обработки пустого запроса.
- **TC-04 Поиск с длинным запросом** – проверка работы с длинными строками.
- **TC-05 Поиск со спецсимволами** – проверка обработки нестандартных символов.
- **TC-08 Поиск с разным регистром (CapsLock)** – проверка нечувствительности поиска к регистру.
### Test Suite: Навигация по сайту
- **TC-06 Переход по результату поиска** – проверка открытия страниц из результатов поиска.
- **TC-09 Переход по разделу “Планеты”** – проверка перехода на страницу выбранной планеты.
### Test Suite: Медиа-контент
- **TC-01 Отображение изображений** – проверка загрузки и отображения изображений.
- **TC-07 Воспроизведение звука** – проверка работы аудиофайлов.
- **TC-10 Проверка загрузки видео** – проверка воспроизведения видеоконтента.
### Test Suite: Интерфейс и взаимодействие
- **TC-11 Работа опроса “Вы хотите в космос?”** – проверка выбора и сохранения ответа пользователя.
- **TC-01 Проверка отображения элементов интерфейса** – проверка корректности загрузки страницы.
